欢迎来到我的个人博客

经过为期5天的不懈努力（P.S.时间用的有点久，因为处女座，你懂得，尝试了各种主题以及对主题的修改），
我的个人博客终于搭建起来了。
之前从来没有接触过这种前端的东西，一直致力于嵌入式软件系统的开发，在接触到git之后，发现利用github的
github page可以用来搭建自己的个人网站，于是乎，果断的按照网上的各种教程依葫芦画瓢的搭建起了这个个人博客，
初期在安装的过程中也遇到一个恼人的问题，导致折腾了一天，在安装好git和nodejs之后，下一步是安装hexo，在输入npm
install -g hexo之后会在全局环境装上hexo，然后执行hexo init初始化博客工作区，npm install在工作区目录下加载所需要
的node module，执行hexo generate，生成静态文件，然后输入hexo server开启本地服务，在浏览器输入localhost：4000
就可以正常访问博客页面了，但是接下来就有一个问题，在_config.yml配置文件中，配置好自己的git仓库之后，
执行hexo deploy一直提示
not found git.就在这个问题上折腾了一整天，查看各种博客都没有关于这个问题的回答
最后通过查看官网hexo.io才发现需要在工作区目录下执行
npm install hexo-deployer-git —save,安装上这个模块之后就可以正常的执行代码的deployer了。
